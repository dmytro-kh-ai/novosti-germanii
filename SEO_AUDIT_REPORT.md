# SEO-аудит novosti-germanii.de

Дата: 2026-07-11

## Реализовано в теме

- Убрана зависимость от внешнего Google Fonts-запроса из `wp_enqueue_scripts`.
- Для featured image новости задано `loading="eager"` и `fetchpriority="high"` для улучшения LCP.
- Для остальных изображений сохранены `loading="lazy"` и `decoding="async"`.
- Автоматически заполняются `alt` и `title` изображений на базе заголовка вложения, если поля пустые.
- Усилены meta description, canonical, Open Graph, Twitter Cards.
- Добавлены `max-image-preview:large`, `max-snippet:-1`, `max-video-preview:-1`.
- Добавлены `hreflang="ru"` и `x-default`.
- Расширена JSON-LD разметка: `NewsArticle`, `Article`, `ImageObject`, `Organization`, `WebSite`, `SearchAction`, `Person`, `BreadcrumbList`.
- Для новостей добавлены `headline`, `image`, `author`, `publisher`, `datePublished`, `dateModified`, `mainEntityOfPage`.
- Добавлены динамические `robots.txt`, `sitemap.xml`, `static-sitemap.xml`, `news-sitemap.xml`, `image-sitemap.xml`.
- Добавлены виртуальные SEO-разделы: `/news/`, `/topics/`, `/categories/`, `/guides/`.
- Добавлены региональные страницы `/topics/.../` для Берлина, Гамбурга, Мюнхена, Кёльна, Франкфурта, Дюссельдорфа, Лейпцига, Дрездена, Штутгарта, Ганновера.
- Добавлены тематические страницы для политики, экономики, миграции, работы, недвижимости, пособий, происшествий, автомобилей, образования, бизнеса, технологий, общества.
- Добавлены виртуальные E-E-A-T страницы: `/o-proekte/`, `/redakcionnaya-politika/`, `/avtory/`, `/kontakty/`, `/istochniki/`.
- Под каждой новостью добавлен блок `Читайте также` до 10 внутренних ссылок.
- Блок `Похожие новости` увеличен до 6 карточек.
- В footer fallback добавлены ссылки на E-E-A-T и evergreen-разделы.
- В sitemap включены важные виртуальные SEO-страницы и topic URLs.

## Ограничения

- Google Search Console, GA4, Google News Publisher Center нельзя подключить без доступа к аккаунтам.
- PageSpeed, Core Web Vitals, индексацию и Discover нельзя достоверно подтвердить без live-доступа к отчетам Google и внешним тестам.
- WebP/AVIF конвертация не реализована на уровне темы. Для этого нужен серверный модуль, CDN или WordPress-плагин оптимизации изображений.
- Виртуальные evergreen-страницы сейчас являются хабами. Для максимального SEO-эффекта нужны отдельные подробные статьи-гайды.
- Профили авторов сейчас базовые. Для E-E-A-T нужны реальные страницы авторов с биографией, специализацией и публикациями.

## Критические ошибки

- Часть городских категорий ранее попадала в общие блоки из-за разных вариантов slug/name. Это исправлено локально.
- Деплой из текущей сессии требует восстановить SSH-ключ или загрузить через GitHub/Hostinger. Без деплоя сайт не получит изменения.
- Если Hostinger Git продолжит деплоить старую версию из GitHub, прямые SSH-изменения могут быть перезаписаны.

## Высокий приоритет

- Задеплоить изменения темы на хостинг.
- Подключить сайт в Google Search Console и отправить `sitemap.xml`, `news-sitemap.xml`, `image-sitemap.xml`.
- Проверить Google News Publisher Center: название издания, язык, страна, разделы, логотипы.
- Настроить GA4 и Search Console linking.
- Проверить PageSpeed Insights после деплоя: mobile LCP, CLS, INP.
- Обеспечить featured images минимум 1200 px для всех новостей.
- Создать полноценные evergreen-гайды: гражданство, ВНЖ, Kindergeld, работа, налоги, авто, страховки, недвижимость.

## Средний приоритет

- Добавить реальные страницы авторов с биографиями.
- Добавить отдельную страницу редакции.
- Расширить страницы `/topics/.../` уникальными текстами 800-1500 знаков.
- Добавить автоматический подбор ссылок между городскими и тематическими разделами.
- Настроить WebP/AVIF на сервере или через CDN.
- Проверить пагинацию архивов и canonical для page/2+.

## Низкий приоритет

- Добавить логотип организации в `Organization` schema.
- Добавить `sameAs` для соцсетей, если они есть.
- Добавить RSS/Atom ссылки для разделов.
- Расширить `image sitemap` caption/license полями при наличии данных.

## Прогноз влияния

- Техническая разметка и sitemaps ускорят обнаружение и понимание страниц поисковиками.
- News sitemap и крупные изображения повышают готовность к Google News.
- Улучшение LCP featured image и удаление внешнего font-запроса должны снизить задержку первого экрана.
- Внутренняя перелинковка повышает crawl depth, распределяет вес по старым материалам и помогает Google находить связанные темы.
- Evergreen-разделы дадут потенциал для стабильного поискового трафика, но эффект будет после наполнения полноценными гайдами.
